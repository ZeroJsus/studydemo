# 安装 leiningen 进行项目管理
https://github.com/technomancy/leiningen
https://codeberg.org/leiningen/leiningen

```bash
curl -O https://raw.githubusercontent.com/technomancy/leiningen/stable/bin/lein
chmod +x lein
mv lein /usr/local/bin/
lein self-install
```
